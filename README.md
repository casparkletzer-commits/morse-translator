# MORSE://TRANSLATOR

A simple, terminal-style **Morse code translator** that runs 100% in your browser — no account, no API key, fully offline-capable.

**Live:** https://casparkletzer-commits.github.io/morse-translator/

## Features
- Text ↔ Morse translation (both directions)
- Play morse as audio (Web Audio API) with adjustable speed (WPM)
- Decode live morse from your **microphone** (requires HTTPS)
- Tap-key input: press & hold to send dots/dashes
- Morse reference table
- Mobile-friendly, green-on-black CRT terminal look

Single self-contained `index.html` — vanilla JS, no build step, no dependencies.
